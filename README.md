PROYECTO FINAL DE DIPLOMADO:



“QUIRQUINCHO SANTO”


	
Plataforma: Multiplataforma

Diplomante: Luis Fernando Mendoza Cáceres

Objetivo General

Desarrollar una aplicación que brinde información actualizada del equipo San José de Oruro


Objetivos Específicos

-	Desplegar información actualizada de noticias del equipo.
-	Generar una tabla de posiciones de todos los torneos donde participa.
-	Desplegar el rol de partidos y resultados de los mismos de todos los torneos donde el equipo participe.
-	Desplegar la plantilla actual de jugadores del equipo.
-	Recibir notificaciones cuando se actualicen los resultados de los partidos.


Librerías y dependencias requeridas.

Para el correcto funcionamiento de la aplicación, se requieren que se instalen las siguientes librerías en la plataforma Ionic:

-	SQLite plugin para el almacenamiento y realización de consultas a SQLite.
-	Librería PouchDB, que nos permite la creación y manejo de la base de datos en el dispositivo.
-	PushPlugin, para la recepción de las notificaciones.
-	angular-websocket, nos provee una interfaz de programación que nos permitirá la recepción de las notificaciones desde el servidor de Ionic.
-	Firebase.js, librería de javascript que nos permitirá la conexión y obtención de obtenidos del servicio de firebase.
